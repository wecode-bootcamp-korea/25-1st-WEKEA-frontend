## 담당 부분 및 상세 내용

- 이상철 : 로그인 및 회원가입 페이지
  - 로그인 페이지 및 회원가입 페이지 레이아웃 구현
  - 로그인 페이지
    - 로그인 유효성 검사 기능 구현
    - input 타입 변경을 통한 비밀번호 보이기 기능 구현
    - DB와의 데이터 통신을 통해 유효한 계정과 비밀번호가 들어왔을 경우의 로그인 기능 구현
    - 로그인 후 토큰 생성 및 저장 구현
  - 회원가입 페이지
    - 회원가입 페이지 유효성 검사에 따른 불일치 화면과 버튼 활성화 기능 구현
    - DB와의 데이터 통신을 통해 회원가입 후 유저 정보를 DB에 저장하는 회원가입 기능 구현
